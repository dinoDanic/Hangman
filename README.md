# React Hangman game

simple react hangman game made with React, Redux, Thunk, Styled Components, Framer Motion and Axios

live: https://admiring-almeida-eefb3c.netlify.app

### Install

    - clone this project
    - cd into the project
    - npm install
    - npm start
